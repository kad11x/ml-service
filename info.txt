How to rerunn repo in linux to update folder

# 1) go to your repo folder (adjust path)
cd ~/Desktop/ml-service

# 2) pull latest code
git pull origin main   # or 'master' if that's your branch

# 3) rebuild the image
docker compose up -d --build        # (or: docker compose up -d --build ml-service)

# 4) check if it worked
docker compose ps
docker compose logs -f
